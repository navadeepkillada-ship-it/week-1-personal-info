# Personal Information Manager

A simple beginner-friendly Python project that stores and displays personal information using variables, user input, loops, and basic calculations.

## Features

* Displays a welcome message
* Stores personal information
* Takes user input for favorite food and color
* Prevents empty input
* Calculates age in months
* Displays a formatted summary
* Beginner-friendly Python concepts

---

## Technologies Used

* Python 3

---

## Concepts Used

This project demonstrates:

* Variables
* Data types
* User input
* `while` loops
* Conditional statements
* String formatting using `f-strings`
* Basic arithmetic operations

## How to Run

1. Install Python 3 on your computer
2. Download or clone this repository
3. Open terminal or command prompt
4. Run the file:

```bash
python personal_info.py
```
